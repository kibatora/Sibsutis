# React Video Platform Demo

Фронтенд переведён с набора статических HTML-страниц на React + Vite + React Router.

## Что внутри

- Каталог видео с добавлением, редактированием и удалением карточек
- Страница просмотра видео с чат-панелью и режимами guest/auth
- Страница регистрации и получения кода доступа
- Локальное хранение состояния через `localStorage`

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```
